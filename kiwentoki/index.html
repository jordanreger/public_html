kiwen toki

the language rock

This is an idea I've had for a little bit to learn languages easier. It involves taking a reference language, in this case toki pona, a target language (see below), and comparing them. The beauty of toki pona is that you can communicate complex ideas with very simple base words; however, you can do the same in any language. For example:

  • toki pona: telo seli pimeja
  • Swedish: varmt svart vatten
  • Japanese: 熱い黒湯 (atsui kuro yu)
  • Korean: 뜨거운 검은 물 (tteugeoun geom-eun mul)
  • Croatian: vruća crna voda
  • Dutch: heet zwart water
  • Spanish: agua negra caliente
  • Hindi: गरम काला पानी
  • English: hot black water

These statements are all pretty vague, but the point is that all of these words are simple descriptors that when strung together describe a more complex idea... coffee, in this case. The point of learning this way is not only to make it easier to learn languages by condensing them, but to make it way easier to learn by talking. If someone asked you for hot black water, especially at a coffee shop, chances are you'd understand what they're talking about. I'd also help them out by giving them the word: coffee. Xiaomanyc has a great video about how this works: "How I Learn Any Language in 24 Hours". This helps you expand your knowledge of the language. What if you want tea? Hot black leaf water. "Oh, you're talking about tea." Congrats! You learned a new word.

I think by having a solid foundation to describe pretty much anything, you can learn any language extremely easy. After all, this is how children learn. I hope to prove this works by using it myself.

Search script

I've built a small script to search through english.json. You can install it by running:

deno install -A -n kiwentoki https://jordanreger.com/kiwentoki/cli.js

Then run ./kiwentoki {word} to search for the toki pona equivalent. You can also uninstall by running:

deno uninstall kiwentoki

This is the start of a translation tool that will eventually be hosted here. At the moment it can only translate English to toki pona, but eventually it will expand to other languages and will allow for translation through toki pona.
